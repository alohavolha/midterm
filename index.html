Got grid?

The grid is a 12-column fluid grid with a max width of 960px, that shrinks with the browser/device at smaller sizes. Here is the code:


        [class^="col-"] {
          width: 100%;
          float: left;
          min-height: 1px;
          padding: 0 1em;
        }

        @for $i from 1 through 12 {
          .col-#{$i} {
            @media screen
              and (min-width: 600px){
              width: (100% / 12) * $i;
            }
          }
        }
